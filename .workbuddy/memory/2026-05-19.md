## 基金评价说明功能（自动生成评语）

### 新增内容
1. **`generateFundCommentary(fund)` 函数**（约6043-6126行）
   - 基于 `calcPerformance()` 的所有指标自动生成3-5句中文评语
   - 评语维度：年化收益评价、风险调整收益（夏普+卡玛）、最大回撤、尾部风险（VaR + Omega）、正收益占比
   - 颜色遵循A股惯例：正收益=红色，负收益=绿色
   - 返回值：包含评语 + "查看完整分析"按钮的 HTML 字符串

2. **`openFullAnalysis(fundId)` 函数**（约6129-6140行）
   - 跳转到归因分析页，自动切换到"净值拟合"Tab
   - 将该基金加入 `selectedNavFitFunds`，触发 `renderNavFit()` 渲染

3. **CSS 样式** `.commentary-box`（约1023行）
   - 浅蓝绿渐变背景，圆角卡片，美观分隔

4. **`renderFundDetailBody` 面板结构修改**
   - 在"绩效分析"和"净值走势"之间插入 `commentaryHtml` 区块
   - 模板插入位置：第7771行 `${commentaryHtml}`

### 实现逻辑
- 评语生成规则（分档）：
  - 年化收益：>20% 优异，>10% 良好，>0 略有正收益，>-10% 略有亏损，其余亏损较大
  - 夏普比率：>2 极佳，>1 优秀，>0 一般，≤0 较差
  - 最大回撤：>-5% 控制非常好，>-10% 较好，>-20% 较大，其余非常大
  - Omega比率：>1.5 上行明显优于下行，>1 上行略优，≤1 下行风险较高
  - 正收益段占比：>60% 趋势性较强，>40% 涨跌参半，≤40% 波动较大
