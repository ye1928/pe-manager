## 私募基金收益计算模型重构

### 背景
用户发现原有公式 `(最新净值 - 成本净值 + 累计分红) × 投资金额` 存在缺陷：
当分红时持仓金额与当前投资金额不同时，用单一投资金额 × 累计分红会导致计算错误。

### 解决方案
彻底重建数据模型，改为**多批次 + 分红事件**结构：

```
基金（fund）
├── 申购批次[]：日期、金额（万元）、成本净值 → 自动算份额
├── 分红事件[]：日期、每份额分红金额
└── 最新净值

收益计算（按批次）：
  份额 = 申购金额 / 成本净值
  资本利得 = (最新净值 - 成本净值) × 份额
  分红收益 = Σ(持有期内每次分红/份额 × 份额)  ← 只统计申购日期之后的分红
  批次总盈亏 = 资本利得 + 分红收益
```

### 新增功能
- 新增申购批次（日期、金额、成本净值，自动算份额）
- 分红事件记录（每份额金额，实时预览各批次收益）
- 基金详情侧滑面板（各批次明细、分红时间线）
- 加权平均成本净值自动计算
