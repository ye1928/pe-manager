## 大师投票席 Bug 修复

### 问题
用户反馈：修改投票系统后「每个模块都进不去」。

### 根本原因
索罗斯 `getReason` 函数中重复声明 `const` 变量，导致 JavaScript 语法错误：
- 第 2817 行已声明 `const isFuturesShort = type === 'futures_short';`
- 第 2864 行又重复声明了同名 `const`
- `const` 在同一作用域内不允许重复声明，浏览器直接拒绝执行整个脚本

### 修复内容
删除了 `c:\Users\52706\WorkBuddy\20260425195840\investment-manager.html` 中第 2864 行的重复声明：
```javascript
// 删除前（错误）：
// 第 2817 行：const isFuturesShort = type === 'futures_short';
// 第 2864 行：const isFuturesShort = type === 'futures_short';  // ❌ 重复声明

// 删除后（正确）：
// 第 2817 行：const isFuturesShort = type === 'futures_short';  // ✅ 唯一声明
// 第 2864 行：if (isFuturesShort) { ... }  // ✅ 直接使用已声明变量
```
